# SpoofDPIDiscord
SpoofDPI’yi kurar ve başlatır. Script, gerekli paketleri kontrol eder, SpoofDPI’yi doğru sürümle yükler ve Discord SpoofDPI proxy üzerinden çalıştırır. (Superonline ve Arch Linux üzerinde test edildi))
- Gerekli dosyalar varsa kurulumu geçer
- Doğrudan Multi-thread üzerinden doğru config ile hem spoofdpi hem discord'u çalıştırır
